# GameAssets

游戏开发**素材来源索引**与合规使用提示：按**人物、场景、贴图材质、UI、音频、特效**等分类整理国内外常用站点链接与说明。

本仓库**不包含**第三方美术/音频/模型文件本体，仅维护文档化索引。适用于独立/原型检索、多引擎团队共享书签与内部版审参照。索引正文见下方；仓库以根目录 [LICENSE](./LICENSE) 为准（MIT），**外链资源的版权与许可由各站点自行约束**。

---

## 游戏素材网站索引（按类别）

本仓库按**人物 / 场景 / 贴图材质**等用途归纳常用来源，并补充 UI、音频、特效与综合商店条目。文中站点经**公开网页与许可页**整理，**不代替法律意见**。

**必读：** 免费≠可任意商用。请逐条查看资源页 **License / 用户协议**，注意 **CC-BY（署名）**、**NC（非商用）**、**SA（相同方式共享）**、**GPL（影响整包分发）** 等限制。国内站点多档会员授权并存，勿只凭「免费下载」判断。

**说明：** 下列 URL 为站点入口；站内资源许可可能因作者、年份、活动而变，以下载页与官网现行条款为准。

## 使用建议（简要）

1. **同一站点多品类**：CG模型网、itch、Fab 等同时覆盖人物与场景，本文分表仅为**便于检索**，不表示站点只属一类。  
2. **扫描摄影类贴图**：若游戏引擎使用虚拟纹理流送或二次处理，仍受原贴图许可约束。  
3. **CC 与 GPL**：角色若带 **GPL** 贴图或代码绑定，可能影响你整体发布策略；独立商业项目优先筛 **MIT/CC0/自研**。  
4. **国内图库**：企业实体发行建议使用带**电子授权书或合同**的套餐，并归档订单记录。
